box-shadow: The box-shadow property adds shadow effects around an element's frame. You can set multiple effects separated by commas. The box-shadow property is specified as a list of shadows, each specified as an offset from the element, blur radius, spread radius, and color.

box-shadow: 10px 10px 5px 5px rgba(0, 0, 0, 0.5);

box-shadow: horizontal-offset vertical-offset blur-radius spread-radius color; 
      /*mutiple shadows*/
      .box-3 {
        box-shadow: 5px 5px 4px 2px rgba(0, 0, 0, 0.3),
          -5px -5px 4px 2px rgb(201, 74, 74, 0.7);
      }
