Short Description

  1. Static: Default position; elements follow normal document flow. No special positioning applied.
  2. Relative: Positioned relative to its normal position.It can be moved from its normal document flow using top, right, bottom, and left.
  3. Absolute: Positioned relative to the nearest positioned ancestor (not static). Removed from normal document flow. parent must be relative.
  4. Fixed: Positioned relative to the viewport. Stays in the same place even during scrolling. Removed from normal document flow.
  5. Sticky: Toggles between relative and fixed, depending on the scroll position. It is treated as relative until it crosses a specified threshold, then it is treated as fixed.
  6. z-index: Controls the vertical stacking order of elements that overlap. Higher values are in front of lower values. Only works on positioned elements (relative, absolute, fixed, sticky).
